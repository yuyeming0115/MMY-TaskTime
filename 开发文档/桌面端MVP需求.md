# MMY-TaskTime 桌面端 MVP 产品需求文档

## 概述
- **版本**：v3.0 Desktop MVP
- **技术栈**：Tauri v2（Rust后端 + 现有HTML/CSS/JS前端）
- **目标**：将Web版封装为原生桌面应用，实现窗口置顶、系统托盘、全局快捷键等桌面端核心能力

## MVP 核心功能

### 1. 窗口置顶（Always on Top）
- 精简模式下默认窗口置顶，不被其他窗口遮挡
- 设置中可开关"置顶"选项
- 完整模式默认不置顶，可手动开启

### 2. 系统托盘（System Tray）
- 任务栏托盘图标显示（闹钟图标⏰）
- 右键菜单：
  - 显示/隐藏主窗口
  - 快速启动常驻任务（5分/10分/30分/40分）
  - 分隔线
  - 退出
- 托盘图标tooltip显示当前运行任务及剩余时间
- 托盘图标闪烁/红点提示任务完成

### 3. 全局快捷键
- `Ctrl+Shift+T`（Win/Linux）/ `Cmd+Shift+T`（Mac）：一键显示/隐藏悬浮窗
- `Ctrl+Shift+M`（Win/Linux）/ `Cmd+Shift+M`（Mac）：切换完整/精简模式
- 快捷键可在设置中自定义（MVP先硬编码）

### 4. 开机自启动
- 设置面板新增"开机自启动"开关
- 使用Tauri的autostart插件实现
- 启动后默认显示精简模式悬浮窗（记住上次位置）

### 5. 边缘吸附收起
- 窗口拖到屏幕边缘（左/右/上）时，自动滑出屏幕只露出3px边缘
- 鼠标悬停在露出的边缘上时，窗口滑出展开
- 鼠标离开窗口后延时1秒自动收起
- 拖离边缘区域恢复正常状态

### 6. 窗口记忆
- 记住窗口位置、大小、模式（完整/精简）
- 下次启动恢复上次状态

## 非目标（MVP不做）
- 自动更新
- 多显示器适配优化（MVP支持但不特殊处理）
- 自定义主题
- 多语言
- 云端同步

## Tauri 配置要点
- 窗口初始大小：360×600（完整模式默认居中）
- 窗口装饰：自定义（无边框，保留自己的拖拽条）
- 透明窗口：精简模式需要透明背景
- 安全：仅允许本地文件系统访问
- 权限：notification、autostart、global-shortcut、tray
