## 开发任务清单

### [x] 任务1：基础架构
- HTML/CSS/JS三文件分离
- 双模式CSS框架（.full-mode / .mini-mode）
- 窗口拖拽功能
- 模式切换功能（↙/↗按钮）
- 实时时钟

### [x] 任务2：多任务数据模型与倒计时引擎
- 任务数据结构设计（id、name、time、loop、running、isPrimary、remaining、total）
- 常驻任务初始化（4个预设）
- 倒计时tick逻辑（每秒更新所有运行中任务）
- 常规任务到点停止 vs 循环任务到点自动重置
- 主任务自动切换逻辑

### [x] 任务3：完整模式UI渲染
- 顶部运行中任务标签栏（多chip显示，点击切换主任务）
- 主任务大卡片（进度条、任务名、大倒计时、停止按钮）
- 4列常驻任务按钮网格（运行状态边框+右上角进度环）
- 自定义任务列表（添加/删除/启动）
- 底部设置/统计按钮

### [x] 任务4：精简模式UI渲染
- 深色毛玻璃背景
- 小时钟（20px）
- 主任务区域（进度条、任务名、34px大倒计时）
- 空闲态显示（"MMY-TaskTime"+"选择任务"）
- 底部4个圆形常驻按钮（带环形进度SVG）
- 当前主任务按钮发光边框

### [x] 任务5：任务交互逻辑
- 点击常驻按钮：启动/停止/切换主任务
- 点击自定义任务：启动/停止
- 点击运行标签：切换主任务
- 停止按钮：停止主任务
- 添加任务表单：名称+时长+循环选项

### [x] 任务6：提醒通知机制
- Notification权限请求
- Web Audio API提示音
- 任务结束通知（区分常规/循环）
- 打卡提醒（9:30/19:30，可自定义时间）
- 打卡每日只提醒一次

### [x] 任务7：数据持久化（localStorage）
- 数据结构设计（customTasks、settings、stats）
- 数据加载（默认值）
- 数据自动保存
- 页面刷新状态恢复

### [x] 任务8：设置面板
- 打卡提醒开关toggle
- 上班/下班时间选择器
- 声音提醒开关
- 通知提醒开关
- 设置数据持久化

### [x] 任务9：数据统计与HTML导出
- 统计数据计算（今日时长、完成数、本周时长）
- 统计面板动态渲染
- 常规任务完成自动记录统计
- AI循环任务不统计
- HTML报告生成与下载

### [x] 任务10：UI细节打磨
- 过渡动画优化
- 按钮hover/active反馈
- 进度条/进度环动画
- 完成状态视觉反馈
- 空状态/空闲态样式
- 模态框动画

### [x] 任务11：测试与修复
- 功能完整性测试 ✓
- 双模式切换测试 ✓
- 多任务并行逻辑验证 ✓
- localStorage持久化验证 ✓
- 提醒功能验证 ✓
