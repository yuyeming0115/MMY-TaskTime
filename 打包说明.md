# 一键打包脚本使用说明

## 文件说明

- `build-win.bat` - Windows 一键打包脚本
- `build-mac.sh` - macOS/Linux 一键打包脚本

## Windows 使用方法

### 方法 1: 双击运行（推荐）
直接双击 `build-win.bat` 文件即可

### 方法 2: 命令行运行
```cmd
build-win.bat
```

## macOS/Linux 使用方法

### 首次使用需要添加执行权限
```bash
chmod +x build-mac.sh
```

### 运行脚本
```bash
./build-mac.sh
```

或者在终端中右键脚本文件选择"打开"

## 打包输出位置

打包完成后，文件会生成在：
- **安装包**: `src-tauri/target/release/bundle/`
  - Windows: `bundle/msi/*.msi` 或 `bundle/nsis/*.exe`
  - macOS: `bundle/dmg/*.dmg` 或 `bundle/macos/*.app`
  - Linux: `bundle/deb/*.deb`, `bundle/appimage/*.AppImage`

- **单文件可执行程序** (Windows):
  - `src-tauri/target/release/mmy-tasktime.exe`

## 前置依赖

### Windows
- Node.js (建议 v18+)
- Rust 和 Cargo (通过 https://rustup.rs/ 安装)
- WebView2 (Windows 10/11 通常已内置)

### macOS
- Node.js (建议 v18+)
- Rust 和 Cargo (通过 https://rustup.rs/ 安装)
- Xcode Command Line Tools (`xcode-select --install`)

### Linux
- Node.js (建议 v18+)
- Rust 和 Cargo (通过 https://rustup.rs/ 安装)
- 系统依赖 (参考: https://tauri.app/v1/guides/getting-started/prerequisites)

## 生成真正的单文件可执行程序

默认情况下，Tauri 会生成安装包。如果需要生成真正的单文件可执行程序（无需安装即可运行）：

### Windows - 生成 Portable 版本

1. 修改 `src-tauri/tauri.conf.json`，在 `bundle` 部分添加：
```json
{
  "bundle": {
    "windows": {
      "nsis": {
        "installMode": "both"
      }
    }
  }
}
```

2. 打包后会在 `src-tauri/target/release/` 下生成 `mmy-tasktime.exe`

### macOS - 打包成单文件

macOS 应用本质上是 `.app` 目录，可以：
1. 直接压缩 `.app` 目录分发
2. 使用 DMG 格式（推荐）
3. 使用第三方工具将 `.app` 打包成真正的单文件

## 常见问题

### Q: 打包失败，提示 "npm" 不是内部命令
**A**: 请先安装 Node.js 并添加到系统 PATH

### Q: 打包失败，提示 Rust/Cargo 未找到
**A**: 请安装 Rust: https://rustup.rs/

### Q: Windows 打包很慢
**A**: 首次打包需要下载和编译依赖，请耐心等待。后续打包会快很多。

### Q: macOS 打包提示 "code signing" 错误
**A**: 如果没有 Apple Developer 证书，可以在 `tauri.conf.json` 中禁用代码签名：
```json
{
  "bundle": {
    "macOS": {
      "signingIdentity": null
    }
  }
}
```

## 自定义配置

如需修改应用名称、版本、图标等，请编辑：
- `package.json` - 应用名称和版本
- `src-tauri/tauri.conf.json` - Tauri 配置
- `src-tauri/icons/` - 应用图标

## 清理构建缓存

如果遇到奇怪的编译错误，可以尝试清理缓存：

**Windows**:
```cmd
npm run tauri build -- --clean
```

**macOS/Linux**:
```bash
npm run tauri build -- --clean
```
